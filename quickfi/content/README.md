# QuickFi Content Directory

**Purpose:** Storage for all QuickFi-related content creation

---

## What Goes Here

- Blog posts and articles
- LinkedIn posts and social media content
- Thought leadership pieces
- Website copy
- Email campaigns
- Press releases
- Case studies
- White papers

---

## Naming Convention

Use descriptive names with dates:
```
YYYY-MM-DD-topic-type.md
```

**Examples:**
- `2026-01-17-embedded-lending-blog-post.md`
- `2026-01-17-linkedin-systems-of-action.md`
- `2026-01-17-investor-deck-narrative.md`

## Quality Checklist

Before finalizing content:
- [ ] References QuickFi's core vision
- [ ] Uses correct terminology (systems of action, embedded, borrower-driven)
- [ ] Avoids incorrect positioning (CRM, incremental, system of record)
- [ ] Includes market context ($1.34T when relevant)
- [ ] Emphasizes transformation (not incremental improvement)
- [ ] Appropriate for target audience
- [ ] Factually accurate with sources
- [ ] On-brand messaging
