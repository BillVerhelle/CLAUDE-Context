## Competitor Categorization Framework

### Category 1: Legacy Systems of Record
**Description:** Traditional core banking, lease accounting, CRMs
**Characteristics:**
- Employee-centric workflows
- Track/store what happened
- Manual processes
- Incremental improvements

**QuickFi Advantage:**
- Fundamental vs. incremental
- Borrower-driven vs. employee-mediated
- Real-time vs. delayed

### Category 2: Workflow Optimization
**Description:** Improve employee efficiency with better tools
**Characteristics:**
- Still employee-centric
- Better UX for staff
- Integration between systems
- Process automation

**QuickFi Advantage:**
- Eliminate employees from workflow entirely
- Borrower self-service
- 24/7 availability

### Category 3: Digital Lending Platforms
**Description:** Online lending but still employee-mediated
**Characteristics:**
- Digital application
- Some automation
- Still requires employee review
- Not embedded

**QuickFi Advantage:**
- Fully autonomous
- Truly embedded
- Complete borrower control

### Category 4: Emerging Systems of Action
**Description:** Building toward QuickFi's vision
**Characteristics:**
- Borrower-driven
- Real-time decisions
- AI-enhanced
- Embedded lending

**QuickFi Advantage:**
- First-mover
- Deeper AI integration
- More complete vision
- Equipment finance focus
